Question-1: What are the different ways to select an element in the DOM?

There are 5 ways to select elements in the DOM.

  • getElementsByTagName()
  • getElementsByClassName()
  • getElementById()
  • querySelector()
  • querySelectorAll()

    • Question-2: What is the difference between innerHTML, innerText, and textContent ?

    "innerHTML" allows us to get or set the HTML content of an element, meaning it recognizes and renders HTML tags. "innerText", on the other hand, retrieves or sets only the visible text inside an element, ignoring hidden elements and trimming extra spaces. Meanwhile, "textContent" gets or sets all text within an element, including hidden elements, but it does not parse HTML tags, treating them as plain text instead.

    Question-3: What is event delegation in the DOM?

    Event delegation is a technique where a parent element handles events for its child elements using a single event listener. This works because events bubble up from child to parent, allowing efficient handling, even for dynamically added elements.

    Question-4: What is event bubbling in the DOM?

    Event bubbling is a process in the DOM where an event starts from the target element and moves upward through its parent elements.

    Question-5: How do you create, add, and remove elements using JavaScript?

    To create an element in JavaScript,we can use document.createElement(). To add an element, we can use methods like appendChild() to place it in the desired location within the DOM. To remove an element, we can use the remove() method or removeChild() on its parent element to remove it from the DOM.
